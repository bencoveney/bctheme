# BCTheme

## Goals:

Consistent set of colours I can use for projects

### Suitable for:

- Web design
  - Info, Success, Error, Warning states
  - Consider white/black forground/background for text
- Terminal colours
- Syntax highlighting
  - Plus diffs
- Game assets
  - Pixel art

### Output as:

- CSS variables
- Terminal colour theme style
- VSCode colour theme
- Aseprite palette
